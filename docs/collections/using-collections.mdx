# Getting started with WaterDB Collections.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

:::tip
I suggest that you start a class and use it, then use that class in commands or UIs.

Example Class Usage:
```js
class UIBuilder {
    constructor() {
        this.db = waterDB.collection('uiBuilder')
    }
}
```
:::

### Syntax:

`waterDB.collection('Example Collection')`:

`Returns waterDBCollection('Example Collection', WorldPersistentStorage())`
